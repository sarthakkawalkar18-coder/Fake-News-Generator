📰 Fake News Headline Generator
📌 Description

The Fake News Headline Generator is a fun Python-based project that generates random, humorous, and fictional news headlines. It combines different subjects, actions, and places to create unique and entertaining headlines every time the program runs.

This project demonstrates the use of basic Python concepts like lists, loops, user input, and the random module, making it a great beginner-friendly project.

🚀 Features
Generates random fake news headlines
Interactive user input (generate multiple headlines)
Uses Python's random module
Lightweight and easy to run
Beginner-friendly code structure

🛠️ Technologies Used
Python 3
Built-in random module

📂 How It Works
The program stores predefined lists of:
Subjects (e.g., celebrities, people, animals)
Actions (e.g., launches, eats, declares war)
Places/Things (e.g., Red Fort, Mumbai Local Train)
It randomly selects one item from each list.

Combines them to form a headline:

BREAKING NEWS: [Subject] [Action] [Place/Thing]
Asks the user if they want another headline.

▶️ How to Run
Make sure Python is installed.

Clone this repository:

git clone https://github.com/your-username/fake-news-generator.git

Navigate to the project folder:

cd fake-news-generator

Run the script:

python main.py

BREAKING NEWS: Virat Kohli eats a plate of samosa

Do you want another headline? (yes/no): yes
